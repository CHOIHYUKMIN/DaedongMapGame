# 말랑말랑 대동맛지도 - 서울경기편

## 프로젝트 개요
한국의 서울/경기 지역을 배경으로 한 위치 기반 퍼즐 RPG 게임입니다.  
귀여운 캐릭터와 함께 전국의 맛집을 여행하며 지역 특산물을 수집하고, 실제 혜택을 받을 수 있는 O2O 연동 게임입니다.

## 게임 특징
- 🗺️ **서울/경기 약 1,000개 지역(동 단위)** 탐험
- 🧩 **3-Match 퍼즐** 메커니즘
- 👤 **10종 캐릭터** (연령/성별 맞춤형)
- 🍖 **지역 특산물 수집** 및 성장 시스템
- 💰 **O2O 맛집 홍보** + 포인트 리워드

## 기술 스택
- **엔진**: Godot Engine 4.2 (오픈소스)
- **개발 언어**: GDScript / C# (선택 가능)
- **데이터**: CSV (로컬 DB)
- **플랫폼**: Mobile (Android / iOS)

## 프로젝트 구조
```
DaedongMapGame/
├── docs/              # 기획 문서
├── data/              # 게임 데이터 (CSV)
├── scripts/           # 핵심 스크립트
└── README.md
```

## 개발 시작하기
1. **Godot Engine 설치**: https://godotengine.org/download
2. 프로젝트 임포트: Godot에서 이 폴더를 열기
3. `docs/Godot_마이그레이션_가이드.md` 참고하여 진행
4. `data/` 폴더의 CSV 파일은 그대로 사용 가능
5. `scripts/`의 C# 코드를 GDScript로 변환하거나 C# 모듈 사용
