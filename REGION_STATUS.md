## 🗺️ 지도 사용 방법

### 지도 조작
- **드래그**: 마우스로 끌어서 지도 이동
- **줌인/아웃**: 마우스 휠 또는 우측 상단 +/- 버튼
- **더블클릭**: 해당 위치 확대

### 지역 선택
1. 지도에서 원하는 지역의 색상 원 클릭
2. 팝업 창에서 지역 정보 확인
3. **서울/부산**: "시작하기" 버튼 클릭
4. **기타 지역**: "🔒 준비 중" 메시지 표시

## ⚠️ 알려진 문제

### 지도가 안 보이는 경우
1. 브라우저 캐시 삭제 (Ctrl+Shift+Delete)
2. 하드 새로고침 (Ctrl+Shift+R)
3. 콘솔(F12) 확인하여 오류 메시지 확인

### 지역 선택이 안 되는 경우
1. 지도가 완전히 로드될 때까지 대기 (3-5초)
2. 콘솔에서 "✅ 지역 선택 지도 초기화 완료" 메시지 확인
3. 색상 원을 정확히 클릭 (원 중앙 부분)
