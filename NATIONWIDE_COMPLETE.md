# 🎊 전국 완성 기념! 대동맛집지도

## 📅 완성 날짜
**2025년 12월 24일** - 대한민국 전국 17개 시도 완성! 🇰🇷

---

## 🎮 게임 개요

**대동맛집지도**는 대한민국 전국을 여행하며 각 지역의 명물 음식과 맛집을 탐험하는 3-Match 퍼즐 게임입니다!

---

## 📊 통계

- 🎮 **총 레벨 수**: 125개
- 🍱 **총 아이템 수**: 200개 이상
- 🏆 **지역별 증표**: 17개 (각 시도별 Legendary 아이템)
- ⭐ **특별 아이템**: 전주 비빔밥, 제주 흑돼지 등

---

## 🎯 게임 특징

### 🧩 3-Match 퍼즐
- 같은 색/모양 3개 이상 매칭
- 다양한 테마의 블록 (각 지역 특색 반영)
- SCORE, COLLECT, BOSS 등 다양한 미션

### 🗺️ 전국 지도
- 17개 시도 모두 선택 가능
- 지역별 컬러와 아이콘으로 구분
- 클릭 한 번으로 지역 이동

### 🍜 맛집 도감
- 각 지역 대표 맛집 정보
- 희귀도 시스템 (C / B / A / SP / Legendary)
- 수집 진행률 표시

### ☁️ 클라우드 저장
- Firebase 실시간 동기화
- Google 로그인 지원
- 여러 기기에서 플레이 가능

---

## 🎨 지역별 특색

각 레벨은 해당 지역의 특색을 반영한 블록 테마를 가지고 있습니다:

- **서울**: 궁궐, 전통시장, 명동 쇼핑가
- **부산**: 바다, 해변, 해산물
- **강원**: 산, 스키, 메밀, 커피
- **제주**: 감귤, 흑돼지, 바다, 한라산

## 📱 플레이 방법

1. **지도에서 지역 선택**
   - 원하는 지역의 컬러 마커 클릭
   - 지역 정보 확인
   - "시작하기" 버튼 클릭

2. **퍼즐 게임 플레이**
   - 블록을 드래그하여 인접 블록과 교환
   - 같은 색/모양 3개 이상 매칭
   - 목표 점수 또는 수집 개수 달성

3. **맛집 아이템 획득**
   - 레벨 클리어 시 해당 지역 맛집 획득
   - 맛집 도감에서 수집 현황 확인
   - 각 지역 완료 시 증표 획득
