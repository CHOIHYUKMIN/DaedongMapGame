# 웹 게임 실행 가이드

## 🎮 게임 실행 방법

### 방법 1: 간단 실행 (권장)
1. `web` 폴더 안의 **`index.html`** 파일을 찾습니다
2. **더블클릭** → 브라우저에서 자동 실행!

### 방법 2: 브라우저에서 직접 열기
1. Chrome / Edge / Firefox 등 브라우저 실행
2. `Ctrl + O` (파일 열기)
3. `d:\DEVELOP\DaedongMapGame\web\index.html` 선택
4. 게임 시작!

---

## 🕹️ 게임 플레이 방법

### 1단계: 메인 메뉴
- **게임 시작** 버튼 클릭

### 2단계: 캐릭터 선택
- 나이대 선택 (10대 / 20대 / 30대 / 40~50대 / 60대+)
- 성별 선택 (남성 / 여성)
- 미리보기에서 캐릭터 확인
- **시작하기** 버튼 클릭

### 3단계: 지도 화면
- 레벨 노드 클릭하여 퍼즐 시작
- 🍴 아이콘 = 맛집 홍보 스테이지 (MP 보너스!)
- 잠긴 레벨은 이전 레벨을 먼저 클리어해야 함

### 4단계: 퍼즐 게임
#### 조작법:
1. **블록 클릭** → 선택 (노란 테두리)
2. **인접 블록 클릭** → 교환
3. **같은 색 3개 이상** 일렬로 배치하면 터짐!

#### 목표:
- 제한된 이동 횟수 안에 목표 점수 달성
- ⭐ 1개 = 기본 클리어
- ⭐ 2개 = 1.2배 점수
- ⭐ 3개 = 1.5배 점수

#### 블록 종류:
- 🎒 빨강 (복주머니)
- 🍯 노랑 (유과)
- 🏮 파랑 (청사초롱)
- 🌲 초록 (소나무)
- 🎭 보라 (부채)

## 📱 모바일에서 플레이

### Android/iOS:
1. PC에서 index.html을 **웹 서버로 호스팅** (예: http://localhost:8000)
2. 핸드폰에서 같은 IP 주소로 접속
3. 터치로 플레이 가능!

### 간단한 로컬 서버 실행 (Python):
```bash
cd d:\DEVELOP\DaedongMapGame\web
python -m http.server 8000
```
→ 핸드폰에서 `http://PC의IP:8000` 접속

## 🔧 문제 해결

### Q: 블록이 안 보여요!
**A**: 브라우저 콘솔 확인 (F12) → 오류 메시지 확인

### Q: 저장이 안 돼요!
**A**: 브라우저의 LocalStorage 사용. 시크릿 모드에서는 저장 안 됨.

### Q: 처음부터 다시 시작하고 싶어요!
**A**: 브라우저 콘솔(F12)에서 `Game.resetGame()` 입력
