# 업데이트 내역

## v0.2 - 드래그 앤 드롭 업데이트 (2025-12-18)

### 🎮 개선된 기능

#### 1. 드래그 앤 드롭 조작
- **이전**: 블록을 두 번 클릭해서 교환
- **현재**: 블록을 드래그해서 원하는 방향으로 스와이프!
  - 위/아래/좌/우 드래그로 인접 블록과 교환
  - 모바일 터치 완벽 지원

#### 2. 화려한 애니메이션
- 타이틀이 통통 튀면서 등장
- 타이틀 글로우 효과 (반짝임)
- 버튼 호버 시 웨이브 효과
- 화면 전환 부드러운 페이드인

#### 3. 퍼즐 애니메이션
- 블록이 회전하며 등장
- 선택 시 펄스(pulse) 애니메이션
- 블록 터질 때 폭발 효과 + 회전
- 파티클 효과 (8방향으로 흩어짐)
- 블록 교환 시 부드러운 스와이프 애니메이션
- 매칭 실패 시 되돌아오는 애니메이션

### 🎯 플레이 방법

1. **블록 선택**: 블록을 마우스/손가락으로 누르기
2. **드래그**: 원하는 방향(위/아래/좌/우)으로 끌기
3. **놓기**: 손을 떼면 인접 블록과 교환
4. **매칭**: 같은 색 3개 이상 연결되면 터짐!
